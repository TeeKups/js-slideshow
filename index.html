Please read these slides and look at HTML for example code.

Do not change the padding and margins of a slide! Create a div inside the slide instead and add padding to it.

Styling of the slide (in slideshow.css) will break otherwise

You can change the color of the arrows by setting the "fill" attribute for class "slide-button".

.slide-button {
    fill: black;
}

You can change the colors of the current slide indicators by setting the following styles

.slideshow-indicator {
    background-color: rgba(0, 0, 0, 0.5)!important;
    border-color: white!important;
}
.slideshow-indicator.active {
    background-color: rgba(255, 255, 255, 0.5)!important;
}
